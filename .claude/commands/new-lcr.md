Listen and Choose a Response (LCR) の新しい問題セットを作成してください。

## 事前準備（自動実行）
1. `docs/topic-history.md` を読み、LCRセクションの使用済みトピックを確認する
2. `day2/` フォルダ内の既存LCRファイルを確認し、次のセット番号を特定する
3. 使用済みトピックと重複しないテーマを選定する

## 指定
$ARGUMENTS
- 例: `campus health center, study abroad office` → 指定されたシーンを含める
- 例: `random` or 空 → 全てランダム生成

## 出力ファイル（Phase 1: スクリプト＋HTML雛形）
1. `day2/lcr-scripts-{N}.md` — 録音用スクリプト＋問題＋Answer Key
2. `day2/lcr-{N}.html` — HTML（音声はプレースホルダー `lcr-q1.mp3` 等で外部参照）

## LCR問題仕様 (v2.2.1)

### 概要
- 約5秒の短い1文を聞く（音声のみ、テキスト表示なし）
- 4つの書かれた選択肢から最も適切な応答を選ぶ
- 8問構成
- キャンパスライフ・日常生活の場面

### 難易度分布
- Q1-Q2: B1（基本的な日常会話、明確な意図）
- Q3-Q5: B1-B2（やや間接的な表現、文脈推測が必要）
- Q6-Q8: B2（暗示的な意味、語用論的理解が必要）

### Response Typology（12タイプ）
- Type 1: 間接的な回答 / Type 2: 情報提供・助言 / Type 3: 共感・同調
- Type 4: 条件付き承諾 / Type 5: 驚き・意外性 / Type 6: 訂正・補足
- Type 7: 代替案の提示 / Type 8: 前提への気づき / Type 9: 意図・計画の表明
- Type 10: 体験の共有 / Type 11: 不確実性の表明 / Type 12: 暗示的な断り

**分布ルール:** 8問で最低6タイプ以上カバー、同じタイプ3回以上禁止

### 話者配置
- Woman と Man を交互に配置（Q1=Woman, Q2=Man, ...）

### 選択肢ルール
- 全選択肢 5〜10語の短い1文
- ダッシュ/セミコロンで文をつなげない
- カンマスプライス禁止（相槌＋本文は許容）
- 最長-最短差が4語以内

### 正解分布
- A=2, B=2, C=2, D=2（完全均等）

### アクセント分布
- American: 3-4問 / British: 1-2問 / Australian: 1-2問 / New Zealand: 0-1問

### 正解の一意性チェック（必須・4段階）
1. 正解は発話に対する自然な応答か
2. 各誤答は明確に不適切か（topic/phonetic/pragmatic/inference trap）
3. 語数の最長-最短差が4語以内か
4. 誤答が別のResponse Typeとして自然な応答になっていないか

### 作成後
1. スクリプト＋問題＋Answer Keyをmdファイルに出力
2. HTML雛形を作成（音声は外部参照プレースホルダー）
3. `docs/topic-history.md` のLCRセクションに使用したトピックを追記
4. 検証サマリーを表示:
   - Response Type分布（6+/12タイプ確認）
   - 正解分布（A=2,B=2,C=2,D=2確認）
   - 難易度分布（B1×2, B1-B2×3, B2×3確認）
   - 選択肢語数チェック（5-10語、差4語以内）
   - 過去セットとの重複なし確認
